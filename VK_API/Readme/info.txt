Доработаем созданный на 1-й лекции мессенджер - реализуем в нём возможность отправки сообщения на стену в соц.сети вконтакте.
На главной странице сайта, предоставляющего какие-либо веб-сервисы, обычно есть ссылка "Разработчикам".
2).vk.com > Разработчикам > Документация > Работа с API > Запросы к API
http://vk.com/dev/api_requests
Для получения ответа в формате XML нужно после имени метода написать .xml.
1).vk.com > Разработчикам > Документация > Список методов: Стена : wall.post.
http://vk.com/dev/wall.post

Данный метод принимает очень много аргументов, однако обязательными являются только:
* owner_id - идентификатор пользователя, на страницу которого нужно добавить запись.
* message - собственно сообщение, которое будет добавленно на стену.
* access_token - токен доступа, специальный ключ, необходимый для аутентификации нашего приложения.

3).Как получить токен ?
vk.com > Разработчикам > Документация > Работа с API > OAuth-авторизация
OAuth - открытый протокол авторизации, который позволяет предоставить третьей стороне ограниченный доступ к защищённым ресурсам пользователя без необходимости передавать ей (третьей стороне) логин и пароль. 
Для получения токена необходимо выполнить переход на https://oauth.vk.com/authorize с аргументами
* client_id - идентификатор нашего приложения. Чтобы приложению был назначен идентификатор, его нужно зарегистрировать.
* redirect_uri - так называемый доверенный адрес - та страница, на которую будет перенаправление после ввода логина и пароля. Если наше приложение - сайт, то это любая страница на нашем сайте. Если нет, как в данном конкретном случае, используем адрес https://oauth.vk.com/blank.html. Доверенный адрес должен быть зарегистрирован при регистрации приложения.
* scope - здесь определяем, на изменение какой информации должны быть права у нашей программы. Ставим wall.
response_type - тип ответа, который нужно получить (ставим token).
* display - как будет выглядеть страница ввода логина и пароля. Ставим popup, в этом случае она откроется во всплывающем окне.

Пример адреса для получения токена
https://oauth.vk.com/authorize?client_id=7148959&redirect_uri=https://oauth.vk.com/blank.html&display=popup&scope=wall&response_type=token

4). Для получения параметра client_id и регистрации redirect_uri нам нужно зарегистрировать наше приложение. 
vk.com > Разработчикам > Документация > Мои приложения.
Задать название, тип Standalone, Подключить приложение.
Вкладка "Настройки". Копируем ID приложения и прописываем Адрес сайта и Базовый домен.
